Веб-приложение для любителей комнатных растений, которое помогает правильно ухаживать за зелеными питомцами.
Возможности:

Интерактивный каталог растений
    Популярные комнатные растения с подробными описаниями
    Детальная информация об условиях содержания
    Категоризация по типам растений

Умный расчет графика полива
    Автоматический подбор оптимальной частоты полива
    Учет 3 параметров: сезон, размер горшка, освещение
    Персонализированные рекомендации по объему воды
    Мгновенный расчет без перезагрузки страницы

Удобный поиск и фильтрация
    Поиск по названию (русскому и латинскому)
    Фильтрация по категориям
    Адаптивный интерфейс для всех устройств
